# # SM Mall REST API

Finals project for CSE1.

## Features
- Flask REST API
- MySQL Database
- CRUD Operations
- JWT Authentication
- XML and JSON output
- Unit Testing

## Setup
1. Create virtual environment
2. Install dependencies
3. Import Bron.sql
4. Run app.py

## Login
POST /login

## Endpoints
- GET /customers
- POST /customers
- PUT /customers/<id>
- DELETE /customers/<id>
- GET /stores
- GET /products
